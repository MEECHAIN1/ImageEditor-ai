### 🔐 ตัวอย่าง `nginx.conf` แบบพร้อมใช้งาน
```nginx
events {}

http {
  server {
    listen 443 ssl;
    server_name rpc.meechain.run.place;

    ssl_certificate /home/runner/workspace/certbot/config/live/rpc.meechain.run.place/fullchain.pem;
    ssl_certificate_key /home/runner/workspace/certbot/config/live/rpc.meechain.run.place/privkey.pem;

    access_log /home/runner/workspace/logs/nginx/access.log;
    error_log  /home/runner/workspace/logs/nginx/error.log;

    location / {
      proxy_pass http://localhost:8080;
    }
  }
}
```

### 🧩 Ritual Commands
- สร้าง log folder:
  ```bash
  mkdir -p ~/workspace/logs/nginx
  ```

- รัน Nginx:
  ```bash
  nginx -c ~/workspace/logs/nginx/nginx.conf
  ```

- ถ้าเจอ `Address already in use`:
  ```bash
  lsof -i :443
  nginx -s reload -c ~/workspace/logs/nginx/nginx.conf
  ```

- ทดสอบ RPC:
  ```bash
  curl https://rpc.meechain.run.place
  ```
